## SUDOCU

**API:** Backend de APIs REST.   
**Gestión:** Frontend de creación y gestión integral de trámites y documentos.  
**MPD:** Frontend del Módulo de Publicación y Digesto.  
**MPC:** Frontend del Módulo de Parametrización y Configuración.  
**Inicio:** Frontend de ingreso al sistema y visualización de notificaciones y novedades.  
**PDF:** Servicio de impresión de PDFs basado en browserless.   
**Cache:** Servicio de cache y sesiones de usuario basado en Redis.   
**SUDOCU Volume:** Volumen de almacenamiento y persistencia de archivos adjuntos de SUDOCU.
